*******************
* Program Control *
*******************

REM
DO / LOOP
FOR var=a TO b [STEP s] / NEXT var
GOSUB label
RETURN
GOTO label
IF expression THEN command [ELSE command] / END IF
REPEAT / UNTIL expression
WHILE expression / WEND
END
